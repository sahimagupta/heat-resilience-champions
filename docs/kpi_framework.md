# KPI Framework — Heat Resilience Champions

**8 Pillars · 46 Indicators · PRE + POST Design**

---

## How to Read This Framework

Each indicator includes:
- **PRE target**: What to measure before the programme (baseline)
- **Benchmark**: Reference value from evidence
- **POST method**: How to collect the outcome data
- **POST timing**: When to measure after intervention
