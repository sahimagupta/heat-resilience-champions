## What This Repo Contains

A complete, open-source impact measurement framework for heat resilience programmes worldwide. Built from 24 peer-reviewed and institutional sources, this repository provides:

| Folder | Contents |
|--------|----------|
| `data/` | 9 CSV datasets + 1 JSON calculator covering mortality, economics, workers, children, urban heat, solutions, and projections |
| `docs/` | KPI framework (8 pillars, 46 indicators), pre/post metrics brief, bibliography, city profiles, research questions |
| `dashboards/` | Five self-contained HTML dashboards — data explorer, impact calculator, K-12 climate education framework, climate schools pre-discussion, and K-12 heat resilience dashboard |

## Key Numbers at a Glance

- **546,000** heat-related deaths annually (Lancet Countdown 2025)
- **$1.09 trillion** in productivity losses (ILO 2024)
- **2.41 billion** workers with high heat exposure (ILO 2024)
- **200 million** children in extreme heat zones (UNICEF 2024)
- **80%** reduction in heat mortality achievable with existing interventions (Lancet Europe 2024)

## How to Use the Data

### For Researchers
All CSVs use standard headers with source attribution in every row. Clone and analyse:
```bash
git clone https://github.com/GlobeFromHome/heat-resilience-champions.git
cd heat-resilience-champions/data
```

### For Programme Designers
1. Open `docs/kpi_framework.md` for the full 46-indicator measurement system
2. Use `docs/pre_post_metrics_brief.md` to design your evaluation
3. Reference `docs/city_profiles.md` for proven intervention models

### For Trainers & Facilitators
1. Open `dashboards/index.html` in any browser — no server needed
2. Use `dashboards/impact_calculator.html` to model programme outcomes
3. Use `dashboards/k12_climate_education_impact_framework.html` for K-12 programme design
4. Use `dashboards/climate_schools_prediscussion_dashboard.html` for pre-workshop data briefing
5. Use `dashboards/k12_heat_resilience_impact_dashboard.html` for heat resilience KPIs and multiplier tracking
6. Pull discussion questions from `docs/questions_bank.md`

### For Funders & Decision-Makers
1. Review `data/economy_impact.csv` for the economic case
2. Check `data/solutions_proof.csv` for evidence of what works
3. Use the impact calculator to model ROI

## Live Dashboard

Open `dashboards/index.html` directly in your browser — it's a fully self-contained HTML file with no external dependencies. Works offline.
